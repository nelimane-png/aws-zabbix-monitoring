# Projet AWS Zabbix Monitoring

## Description
Mise en œuvre d'une infrastructure cloud de supervision centralisée
basée sur Zabbix, Docker et AWS.

## Architecture
- 1 serveur Zabbix (Ubuntu + Docker)
- 1 client Linux (Ubuntu)
- 1 client Windows (Windows Server)

## Technologies utilisées
- AWS EC2 / VPC
- Zabbix
- Docker & Docker Compose
- Ubuntu 22.04
- Windows Server 2022
